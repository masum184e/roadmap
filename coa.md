# Topics to Learn for CoA

## 1. Introduction to Computer Organization

- Basic computer components (CPU, Memory, I/O, Bus)
- Von Neumann vs. Harvard architecture
- Instruction cycle and execution steps
- Performance metrics: latency, throughput, CPI, MIPS

## 2. Data Representation

- [Number systems: binary, octal, decimal, hexadecimal](#)
- [Signed number representation: 1’s & 2’s complement](#)
- Floating-point representation (IEEE 754)
- Character encoding (ASCII, Unicode)
- Arithmetic operations on integers and floating-point numbers
- Overflow detection and handling

## 3. CPU Organization

- CPU structure: ALU, CU, registers
- Register organization: general-purpose, special-purpose
- Status and flag registers
- Micro-operations and timing
- Instruction pipelining (basic concept)

## 4. Instruction Set Architecture (ISA)

- Instruction formats (R, I, J types, etc.)
- Addressing modes (immediate, direct, indirect, indexed, etc.)
- Types of instructions (data movement, arithmetic, control, logical)
- Assembly language basics
- RISC vs. CISC architectures
- Example architectures: MIPS, ARM, x86

## 5. Arithmetic and Logic Unit (ALU)

- Combinational circuits: adder, subtractor, multiplier
- Carry lookahead adder
- Booth’s algorithm and binary multiplication
- Division algorithms
- Floating-point arithmetic
- Logic unit design (AND, OR, NOT, XOR operations)

## 6. Control Unit Design

- Hardwired control vs. Microprogrammed control
- Control word, microinstruction, and control memory
- Sequence control, branch control
- Timing and synchronization

## 7. Memory Organization

- Memory hierarchy: registers → cache → main → secondary
- Cache memory: mapping techniques (direct, associative, set-associative)
- Cache replacement policies (LRU, FIFO, random)
- Virtual memory and paging
- Memory interleaving and access time
- DRAM vs. SRAM, ROM, EEPROM

## 8. Input/Output Organization

- I/O mapped vs. memory mapped I/O
- Programmed I/O, Interrupt-driven I/O, and DMA
- Interrupt structure and priority
- I/O interfaces and controllers
- Bus arbitration and protocols

## 9. Pipelining and Parallelism

- Instruction pipeline stages
- Hazards: data, structural, control
- Techniques to reduce hazards (stalling, forwarding, branch prediction)
- Superscalar and VLIW concepts
- Introduction to parallel processing and SIMD/MIMD

## 10. Multiprocessor and Multicore Systems

- Shared memory vs. distributed memory systems
- Cache coherence and consistency
- Interconnection networks
- Synchronization mechanisms

## 11. Performance and Benchmarking

- CPI, MIPS, MFLOPS
- Amdahl’s Law
- Benchmarking tools and workloads

## 12. Emerging Topics (Advanced / Optional)

- GPU architecture basics
- RISC-V instruction set
- Low-power processor design
- Hardware security fundamentals
- Quantum and neuromorphic architectures (intro level)
